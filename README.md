# Localize-me

Localize-me is an app created to show the weather data based on your location.

![image](https://user-images.githubusercontent.com/57424426/145696018-95973a40-d140-4444-b3a0-07a23a9569c9.png)

## Instalation

Install all dependencies in the project directory by the following command:

### `yarn`

## .env

Create a file called `.env` based on `.env.example` with your credentials in the project directory.

## Running the app

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

## Running Tests

### `yarn test`

Launches the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.
